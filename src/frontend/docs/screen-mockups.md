# Screen Mockups

This document provides wireframe descriptions for the main screens in the Swaraj Infotech application.

## Design System
- **Theme**: Dark background with orange gradient accents
- **Style**: Glassmorphic cards with blur effects
- **Typography**: Space Grotesk for headings, Inter for body
- **Colors**: Orange primary (#FF8C42 range), dark backgrounds
- **Animations**: Subtle hover effects, smooth transitions
